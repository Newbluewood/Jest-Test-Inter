# Jest-Test-Inter

**TypeScript** vežbe i testovi sa **Jest**‑om — materijal za praksu / stažiranje.

## Pregled

- `src/app/` – izvorni kod aplikacije  
- `src/test/` – Jest testovi  
- `jest.config.ts` – konfiguracija Jest‑a  
- `tsconfig.json` – TypeScript  

## Zahtevi

- Node.js (LTS preporučen)  
- `npm`  

## Pokretanje testova

```bash
npm install
npm test
```

## Stack

- TypeScript 5.x  
- Jest 29.x  
- ts-jest, ts-node, @types/jest
